# 🌾 AgroSarthi Models
> Machine learning models for intelligent agricultural decision support.

---

## 🚀 Overview
**AgroSarthi Models** is a collection of machine learning models designed to assist in agricultural decision-making.

It integrates multiple prediction systems to provide insights related to crop selection, disease detection, yield estimation, and soil analysis.

---

## 🧠 Key Features
- 🌱 Crop recommendation system  
- 🌿 Disease prediction model  
- 📈 Yield prediction  
- 🌍 Soil feature analysis  

---

## 🛠 Tech Stack
- Python  
- Machine Learning / Deep Learning  
- Pandas, NumPy  
- TensorFlow / Scikit-learn (update if needed)
